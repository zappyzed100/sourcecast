# licenses/ — ライセンス原文

読み辞書のデータソース（`config/readings/sources.yaml`）のうち、原文保持が
条件のライセンス（Apache-2.0のSudachiDict等）の**ライセンス全文**をここに置く
（development-plan.md §8.3）。

- ファイル名は `<source_id>-LICENSE.txt`（例: `sudachidict-LICENSE.txt`）
- 各辞書の**取得スクリプトが取得時に配布物からコピーする**——手書き・Web貼り付けをしない
  （配布物同梱の原文が正本。取得スクリプトは §8.4 のソース別タスクで実装）
- 出典・ライセンスの一覧は [THIRD_PARTY_NOTICES.md](../THIRD_PARTY_NOTICES.md)（機械生成）
